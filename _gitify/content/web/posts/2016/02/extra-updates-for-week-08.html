                  Our loyal Release Robot Robbie has compiled a list of new and updated MODX Extras in the week of February 22, 2016.

                  • FileDownload R 2.0.0-beta2, released on 2016-02-22 View Changelog

                    ×

                    FileDownload-R 2.0.0-beta2 (February 22, 2016)

                    ============================================

                    • + removing old data after converting

                    FileDownload-R 2.0.0-beta1 (February 19, 2016) ============================================

                    • + fixed anti hotlink on FURL
                    • + [#8][#28][#33] Fixed unicode problems
                    • + [#42] show empty folder
                    • + [#43] update build of fd_count, convert database
                    • + [#44] fixed saltText comparison
                    • + [#47] fixed duplicate items on Group By Directory
                    • + rename folder
                    • + [#17][#29] add media source support
                    • + add geolocation support

                  • virtuNewsletter 2.0.0-beta3, released on 2016-02-22 View Changelog

                    ×

                    virtuNewsletter 2.0.0-beta3 (February 22, 2016)=================================================

                    • - avoid subscriber to be re-queued back to the past newsletters when their info is updated

                    virtuNewsletter 2.0.0-beta2 (February 22, 2016)

                    =================================================

                    • - add boolean option to not requeue existing subscriber on syncing
                    • - show update panel after create new newsletter
                    • - fix addSubscriberQueues() to check existing queue
                    • - add search box to Reports grid
                    • - add update subscriber feature on Subscribers page
                    • - add more info on newsletter's grid
                    • - refresh newsletter's grid when form is saved
                    • - fixed empty email on sync

                  • FileAttach 1.0.6-pl, released on 2016-02-25 View Changelog

                    ×

                    1.0.1

                    ==============

                    - Updated FileItem model

                    - UI editor rearrange

                    1.0.2

                    ==============

                    - Add download option

                    - Add rank field in model

                    - Add ability to manually rearrange order

                        - Add ability to delete files

                        - Display thumbnails

                    1.0.4

                    ==============

                    - DB scheme enhance

                    - Add SQLSrv support

                    - Fix remove method

                    1.0.5

                    ==============

                    - Add preset to allow anonymous download

                    - Fix snippet arguments typo

                    1.0.5-pl1

                    ==============

                    - Add `ext` placeholder for extension

                    - Add showHASH to snippet sheet

                    - Fix hash loading

                    1.0.6

                    ==============

                    - Add frontend processors

                    - Add new policy for frontend uploader
