          Our loyal Release Robot Robbie has compiled a list of new and updated MODX Extras in the week of April 04, 2016.

          • Updater 0.3.9-beta, released on 2016-04-05 View Changelog

            ×

            UPDATER - the universal update notifier

            See forum thread for basic support and more information at http://forums.modx.com/thread/?thread=96613

            0.3.9-beta (05.04.2016)

            • fix showing update available if the only possible update has been filtered out
            • fix recoverable error in log (thanks @sotwell)
            • add clear cache event to clear also updater's internal cache (not only on package changes)
            • fixed missing lexicon entries
            • changed lexicon to namespaced lexicon
            • fixed bug with caching lexicon entries and log messages
            • compressed inline css

          • ContentBlocks 1.4.1-pl, released on 2016-04-05 View Changelog

            ×
            • Only hide delete button on repeater fields [#380]
            • Fix E_NOTICE caused by lexicon files
            • Fix the template builder being broken since 1.4 [#385]
            • Hide filter options on subfield grids [#383]
            • Fix ID conflicts in the subfield grids when editing a repeater
